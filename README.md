# 🔐 Tailwind Login Page

Uma página de login simples e responsiva desenvolvida com **Tailwind CSS**, com foco em praticar layout, responsividade e estilização moderna.

---

## 📸 Preview

![preview](https://raw.githubusercontent.com/guitrinda/tailwind-login-page/blob/main/src/assets/preview.png)

---

## 🚀 Tecnologias utilizadas

* HTML5
* Tailwind CSS

---

## 🎯 Objetivo do projeto

Este projeto foi criado com o objetivo de:

* Praticar o uso de classes utilitárias do Tailwind
* Trabalhar com responsividade (mobile-first)
* Criar uma UI limpa e moderna
* Melhorar organização de layout com Flexbox

---

## 💡 Funcionalidades

* Layout responsivo (mobile e desktop)
* Efeitos de hover
* Inputs estilizados com foco
* Estrutura pronta para integração futura com API

> ⚠️ Este projeto não possui integração com backend (somente front-end)

---

## 📱 Responsividade

* 📱 Mobile: layout em coluna
* 💻 Desktop: layout dividido em duas seções

---

## 🛠️ Como rodar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/guitrinda/tailwind-login-page.git
```

2. Abra o projeto:

```bash
cd tailwind-login-page
```

3. Execute o Tailwind:

```bash
npx tailwindcss -i ./style/style.css -o ./style/output.css --watch
```

4. Abra o `index.html` no navegador
